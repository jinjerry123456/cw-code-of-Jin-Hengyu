
        Transformer模型使用指南:
        
        1. 安装必要的库:
           pip install transformers torch
        
        2. 运行以下代码:
           from transformers import pipeline
           sentiment_analyzer = pipeline("sentiment-analysis")
           
           # 分析评论
           results = sentiment_analyzer("这个药物非常有效，没有副作用")
           print(results)  # 输出情感标签和置信度
